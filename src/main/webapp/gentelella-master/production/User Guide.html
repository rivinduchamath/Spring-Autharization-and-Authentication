Port Number "2000" - Authentication and Authorization JWT

Initially One user in the system name : Kamal / userName : Kamal / Password : 12345 / Role : ROLE_ADMIN

There is another role In System ROLE_USER

ADMIN_USER > Kamal Can add ROLE_USER to the System

    Before adding User Role Kamal Must Log in to the system to get access_token

  • Kamal Must log in to the system URL : http://localhost:2000/api/login :: Request Type : POST
  • Go to Postman : Body >> x-www-form-urlencoded
  • key :: username : VALUE :: Kamal
  • key :: password : VALUE :: 12345

    Next ADMIN_USER kamal can add User Roles to the system

  • Kamal Must Pass the access_token in header
  • Go to Postman : Headers
  • key :: Authorization : VALUE :: Bearer space Token (E.g.eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9........)
  • Inside the Body He Can Pass User Data as json Object
  • URL : http://localhost:2000/api/user/save Type : Post
  •    {
       "name": "AMri",
       "userName": "nimal123",
      "password": "1866",
       "roles": [

                            "id": 3,

                         }
          ]
       }

    ADMIN_USER and ROLE_USER can Get all Users in the System

  • User Must Pass the access_token in header
  • Go to Postman : Headers
  • key :: Authorization : VALUE :: Bearer space Token (E.g.eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9........)
